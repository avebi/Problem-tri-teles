Navod:
1. Pro spuštění programu se v konzoli objeví nabýdka na možné nastavení. Vybrat se dá mezi vlastním (manuálním) nastavením, náhodným kdy program sám vybere veličiny a přednastavenými něčím zajímavými konstelacemi (kvůli malé vzdálenosti Měsíce od Země vůči vzdálenosti Země od Slunce není Měsíc příliš zřetelný, ale zazoomováním na Zemi se objeví i Měsíc, který ho obíhá).
2. Při vybrání náhodného či přednastaveného režimu se již o nic víc nemusíte starat až na družici o které se zmíním ještě později.
3. Při vlastním režimu vám budou postupně pokládané otázky k hmotnosti, počáteční rychlosti a počátečné poloze každého tělesa. Při zadávání rychlosti a polohy používejte mezerník pro oddělení jednotlivých hodnot například takto: 2.3 4.4 2.1 Vždy je také možnost zadat r pro vybrání náhodné hodnoty.
4. Poslední otázkou je jestli chcete přidat družici, tedy těleso s hmotností 0, na kterou tím pádem gravitačně působí ostatní  tělesa, ale ona sama na ostatní tělesa nepůsobí. Družice je vždy znázorněna tečkovanou čárou.
5. Po zadání všech požadovaných údajů se v jiném okně objeví animace, která ukazuje průbežný pohyb těles a jeden obrázek, kde jsou zachycené trajektorie.
